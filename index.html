Control de Gastos Diarios con Alertas

Aplicación web para ayudar a personas con ingresos ajustados a gestionar sus gastos de forma clara, visual y eficiente.

Descripción del Problema

¿Por qué es necesaria esta solución?

En el día a día, es muy común gastar dinero en cosas pequeñas sin darnos cuenta. Pasajes, comidas rápidas, antojos, salidas o suscripciones parecen inofensivas, pero al repetirse constantemente terminan afectando nuestro bolsillo más de lo que imaginamos. Muchas veces recién al final del mes notamos que el dinero no alcanzó, generando preocupación y estrés.

La mayoría de las personas no cuenta con una herramienta sencilla que les ayude a ver claramente en qué se va su dinero. Sin un registro fácil y visual, resulta difícil identificar hábitos de gasto o saber en qué se podría ahorrar, lo que impide tomar decisiones a tiempo.

Este proyecto busca ofrecer una solución tecnológica que permita registrar, visualizar y analizar los gastos diarios mediante alertas claras.

Herramientas del proyecto

React

React es una biblioteca de JavaScript diseñada para crear interfaces de usuario dinámicas, rápidas y altamente interactivas, especialmente en aplicaciones web modernas. Se basa en la idea de construir la interfaz como una composición de componentes reutilizables, cada uno responsable de su propio estado y comportamiento.

Gracias a su enfoque declarativo y al uso del Virtual DOM, React optimiza las actualizaciones de la interfaz, logrando experiencias fluidas incluso en aplicaciones complejas. Es ideal para desarrollar Single Page Applications (SPA), paneles administrativos, plataformas escalables y productos centrados en la experiencia del usuario.

NestJS

NestJS es un framework backend progresivo para Node.js enfocado en la creación de aplicaciones robustas, escalables y mantenibles. Está construido sobre TypeScript y se inspira en patrones de arquitectura consolidados como MVC, inyección de dependencias y programación modular, lo que facilita el desarrollo de sistemas complejos desde el inicio.

Proporciona una estructura clara y opinada que ayuda a organizar el código de forma consistente, ideal para APIs REST, GraphQL, microservicios y aplicaciones empresariales. Además, se integra de forma nativa con librerías populares como Express o Fastify, ofreciendo alto rendimiento sin sacrificar productividad.
